# Installer note for anyone that uses it.
The rc's and wallpapers thing extracts the wallpaper to config, and just turn on Hidden Files (Ctrl+H), and then, right click (on Golden_Dark_6k.png) when you click on config, and click Set as... and then Set as wallpaper.
